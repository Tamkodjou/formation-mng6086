Intelligence émotionnelle au service du leadership tech
Cible : Chefs d'équipe de développeurs chez CGI (promus récemment depuis des postes techniques) 
Angle : Excellents techniciens devenus gestionnaires, ils manquent de compétences relationnelles. Décalage fréquent entre savoir-faire technique et savoir-être managérial. 
Modules potentiels : Conscience de soi, régulation émotionnelle, empathie dans un contexte agile, gestion des conflits entre développeurs.
